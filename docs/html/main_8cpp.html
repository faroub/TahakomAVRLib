Detailed Description

Demo example: Blink a Led.

Usage demonstration of the TahakomAVRLib in a simple blink a Led example. This example demonstrates the use of the Led and Pin abstraction objects.

  • Compiler: gcc-avr
  • Supported devices: The example compiles on the ATmega48P/88P/168P/328P AVR family
