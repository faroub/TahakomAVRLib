\doxysubsection{Detailed Description}
Demo example\+: Blink a Led. 

Usage demonstration of the Tahakom\+A\+V\+R\+Lib in a simple blink a Led example. This example demonstrates the use of the Led and Pin abstraction objects.


\begin{DoxyItemize}
\item Compiler\+: gcc-\/avr
\item Supported devices\+: The example compiles on the A\+Tmega48\+P/88\+P/168\+P/328P A\+VR family
\end{DoxyItemize}
